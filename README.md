#  Training Center Database

## About

This project is a training center database built using MySQL. It manages students, instructors, courses, and enrollments. The main goal of this project is to apply the SQL concepts that I learned during the Database course.

## What I Used

- MySQL
- MySQL Workbench

## Project Features

- Create and manage database tables
- Insert, update, delete, and retrieve data (CRUD)
- Use Primary Key and Foreign Key relationships
- Apply constraints (NOT NULL, UNIQUE, DEFAULT, CHECK)
- Use String Functions
- Use Aggregate Functions
- Filter and sort data
- Modify tables using ALTER TABLE
- Perform INNER JOIN, LEFT JOIN, and RIGHT JOIN
- Create and use a VIEW
- Use WITH ROLLUP
- Create a Stored Procedure (Bonus)

## Database Tables

- Students
- Instructors
- Courses
- Enrollments

## How to Run

1. Open the SQL file in MySQL Workbench.
2. Run the script from the beginning.
3. The database and all tables will be created automatically.
